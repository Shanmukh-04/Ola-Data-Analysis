# 🚖 OLA Ride Booking Data Analysis – Power BI Project

## 📌 Project Overview

This project presents an End-to-End Data Analysis of OLA ride booking data using SQL, Excel, and Power BI.

The objective of this project is to analyze ride booking patterns, revenue trends, cancellations, vehicle performance, and customer/driver ratings to generate meaningful business insights.

An interactive Power BI dashboard with 5 analytical tabs was developed to help stakeholders make data-driven decisions.

---

## 🛠️ Tools & Technologies Used

- SQL – Data querying & business analysis
- Excel – Data cleaning & preprocessing
- Power BI – Data visualization & dashboard development

---

## 📊 Dashboard Structure

The Power BI report consists of 5 interactive tabs:

---

### 1️⃣ Overall Analysis

**Key Visuals:**
- Ride Volume Over Time
- Booking Status Breakdown
- Total Bookings KPI
- Total Booking Value KPI

**Purpose:**
Provides a high-level overview of ride trends and booking performance.

---

### 2️⃣ Vehicle Type Analysis

**Key Visuals:**
- Top 5 Vehicle Types by Ride Distance
- Total Booking Value by Vehicle Type
- Average Distance Travelled
- Total Distance Travelled

**Purpose:**
Compares vehicle category performance and identifies high-performing segments.

---

### 3️⃣ Revenue Analysis

**Key Visuals:**
- Revenue by Payment Method
- Top 5 Customers by Total Booking Value
- Ride Distance Distribution Per Day

**Purpose:**
Analyzes revenue streams and customer contribution patterns.

---

### 4️⃣ Cancellation Analysis

**Key Visuals:**
- Cancelled Ride Reasons (Customer)
- Cancelled Ride Reasons (Driver)
- Cancellation Rate KPI

**Purpose:**
Identifies major causes of cancellations and operational gaps.

---

### 5️⃣ Ratings Analysis

**Key Visuals:**
- Driver Ratings by Vehicle Type
- Customer Ratings by Vehicle Type

**Purpose:**
Evaluates service quality and customer satisfaction levels.

---

## 📈 Key Business Insights

- Majority of rides are successfully completed.
- Certain vehicle types generate higher booking value.
- Cash and UPI contribute significantly to revenue.
- Cancellation reasons highlight operational improvement areas.
- Ratings indicate consistent service quality across categories.

---

## 🎯 Business Recommendations

- Introduce driver incentives during peak hours.
- Reduce cancellations by addressing key customer and driver concerns.
- Focus marketing on high-revenue vehicle categories.
- Reward top-rated drivers to maintain service excellence.
